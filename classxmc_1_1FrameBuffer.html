◆ endRender()

XmcStatus xmc::FrameBuffer::endRender ( bool  transferToDisplay = true)

End rendering to the frame buffer. If transferToDisplay is true, this function will initiate a transfer of the back buffer to the display. If double buffering is enabled and a transfer is already in progress, this function will wait for the current transfer to complete before starting a new one.

Parameters
transferToDisplayIf true, the back buffer will be transferred to the display after rendering is complete.
Returns
An XmcStatus code indicating success or failure of the operation
