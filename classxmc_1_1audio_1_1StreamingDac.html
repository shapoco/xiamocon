Detailed Description

Streaming DAC class. This class provides an interface for a streaming DAC, which can be used to output audio data in real-time.

XmcStatus xmc::audio::StreamingDac::service ( )

Service the SDAC. This should be called periodically to allow the SDAC to process audio data and output it. The SDAC will call the requestData callback of the source port as needed to get more audio data to output.
