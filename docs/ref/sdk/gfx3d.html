3D グラフィックス API¶

概要¶

3D 描画は次の流れで使用します。

  1. Graphics3D を生成して描画先を設定する

  2. Scene3D / Node3D / Mesh3D / Primitive3D を組み立てる

  3. beginRender() → render() → endRender() で 1 フレーム描画する
