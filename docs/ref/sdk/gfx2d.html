2D グラフィックス API¶

概要¶

2D 描画は主に次の 3 つの要素で構成されます。

  • Sprite: 画像バッファ

  • Graphics2D: Sprite への描画操作

  • FrameBuffer: 画面転送を含む描画フレーム管理
