гғһгғ«гғҒгӮігӮў APIпғҒ

жҰӮиҰҒпғҒ

гғһгғ«гғҒгӮігӮў API гҒҜгҖҒгӮ»гӮ«гғігғҖгғӘгӮігӮў (core1) дёҠгҒ§гӮҝгӮ№гӮҜй–ўж•°гӮ’еҸҚеҫ©е®ҹиЎҢгҒҷгӮӢгҒҹгӮҒгҒ® API гӮ’жҸҗдҫӣгҒ—гҒҫгҒҷгҖӮ

core1 гҒ§е®ҹиЎҢгҒҷгӮӢй–ўж•°гҒҜгҖҒжҲ»гӮҠеҖӨгҒ§з¶ҷз¶ҡеҸҜеҗҰгӮ’еҲ¶еҫЎгҒ—гҒҫгҒҷгҖӮ

еһӢпғҒ

xmc::Core1TaskFuncпғҒ

using xmc::Core1TaskFunc = bool (*)();

core1 гҒ§е®ҹиЎҢгҒҷгӮӢгӮҝгӮ№гӮҜй–ўж•°гҒ®еһӢгҒ§гҒҷгҖӮ

й–ўж•°гҒҢ true гӮ’иҝ”гҒҷгҒЁж¬ЎгҒ®еҸҚеҫ©гҒёйҖІгҒҝгҖҒfalse гӮ’иҝ”гҒҷгҒЁ core1 гҒ®гӮҝгӮ№гӮҜгғ«гғјгғ—гӮ’зөӮдәҶгҒ—гҒҫгҒҷгҖӮ

гӮ°гғӯгғјгғҗгғ«й–ўж•°пғҒ

xmc::startCore1пғҒ

XmcStatus xmc::startCore1(xmc::Core1TaskFunc task);

жҢҮе®ҡгҒ—гҒҹгӮҝгӮ№гӮҜй–ўж•°гӮ’ core1 гҒ§иө·еӢ•гҒ—гҒҫгҒҷгҖӮ

task гҒҜ core1 дёҠгҒ§з№°гӮҠиҝ”гҒ—е‘јгҒіеҮәгҒ•гӮҢгҒҫгҒҷгҖӮ гҒҷгҒ§гҒ« core1 гҒҢиө·еӢ•жёҲгҒҝгҒ®е ҙеҗҲгҒҜеӨұж•—гҒ—гҖҒгӮЁгғ©гғјгӮігғјгғүгҒҢиҝ”гҒ•гӮҢгҒҫгҒҷгҖӮ

жҲ»гӮҠеҖӨгҒҜ XmcStatus гҒ§гҖҒжҲҗеҠҹжҷӮгҒҜ XMC_OK гҒ§гҒҷгҖӮ

xmc::stopCore1пғҒ

XmcStatus xmc::stopCore1(uint32_t timeoutMs = 1000);

еӢ•дҪңдёӯгҒ® core1 гӮ’еҒңжӯўгҒ—гҒҫгҒҷгҖӮ

timeoutMs гҒ«гҒҜеҒңжӯўе®ҢдәҶгӮ’еҫ…гҒӨжңҖеӨ§жҷӮй–“ [ms] гӮ’жҢҮе®ҡгҒ—гҒҫгҒҷгҖӮ еҒңжӯўиҰҒжұӮеҫҢгҖҒгӮҝгӮ№гӮҜгҒҢзөӮдәҶгҒ—гҒӘгҒ„е ҙеҗҲгҒҜгғ—гғ©гғғгғҲгғ•гӮ©гғјгғ е®ҹиЈ…гҒ«еҝңгҒҳгҒҰ core1 гҒҢеј·еҲ¶зҡ„гҒ«гғӘгӮ»гғғгғҲгҒ•гӮҢгҒҫгҒҷгҖӮ

жҲ»гӮҠеҖӨгҒҜ XmcStatus гҒ§гҒҷгҖӮжҲҗеҠҹжҷӮгҒҜ XMC_OKгҖҒеҒңжӯўеӨұж•—жҷӮгҒҜгӮЁгғ©гғјгӮігғјгғүгҒҢиҝ”гҒ•гӮҢгҒҫгҒҷгҖӮ
