◆ xmc_i2c_start_transaction()

xmc_status_t xmc_i2c_start_transaction ( )

Start an I2C transaction.

This function must be called before accessing I2C devices or changing I2C peripheral settings. I2C transactions from other tasks will be blocked until xmc_i2c_end_transaction is called.

Returns
XMC_OK if the transaction was successfully started.
