タイマー API
################################################################################

概要
================================================================================

タイマー API では、周期コールバックを行う `RepeatingTimer` と、
時刻取得・スリープ関数を提供します。

型
================================================================================

xmc::TimerCallback
--------------------------------------------------------------------------------

.. code-block:: cpp

	using xmc::TimerCallback = bool (*)(void *context);

周期タイマーのコールバック型です。
戻り値が `true` の場合は継続、`false` の場合はタイマーを停止します。

RepeatingTimer
================================================================================

xmc::RepeatingTimer::startMs
--------------------------------------------------------------------------------

.. code-block:: cpp

	XmcStatus xmc::RepeatingTimer::startMs(uint32_t intervalMs, xmc::TimerCallback cb, void *context);

指定周期 [ms] でコールバックを呼び出すタイマーを開始します。

xmc::RepeatingTimer::cancel
--------------------------------------------------------------------------------

.. code-block:: cpp

	void xmc::RepeatingTimer::cancel();

動作中タイマーを停止します。

時刻取得
================================================================================

xmc::getTimeMs
--------------------------------------------------------------------------------

.. code-block:: cpp

	uint64_t xmc::getTimeMs();

現在時刻をミリ秒単位で取得します。

xmc::getTimeUs
--------------------------------------------------------------------------------

.. code-block:: cpp

	uint64_t xmc::getTimeUs();

現在時刻をマイクロ秒単位で取得します。

エポックは実装依存ですが、単調増加する時刻として使用できます。

待機
================================================================================

xmc::sleepMs
--------------------------------------------------------------------------------

.. code-block:: cpp

	void xmc::sleepMs(uint32_t ms);

指定ミリ秒だけスリープします。

xmc::sleepUs
--------------------------------------------------------------------------------

.. code-block:: cpp

	void xmc::sleepUs(uint32_t us);

指定マイクロ秒だけスリープします。
