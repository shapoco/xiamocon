◆ render()

void xmc::Mixer::render ( int16_t *  buffer,
uint32_t  num_samples 
)

Renders the mixed audio data into the given buffer. This is called by the audio system when it needs more audio data. The buffer is an array of 16-bit signed integers, and num_samples is the number of samples to render. The mixer should fill the buffer with the mixed audio data from all sources. The mixer should not write more than num_samples samples to the buffer.

Sets the audio source for a given index. The mixer will mix together all sources that are set. The index must be between 0 and the number of sources specified in the constructor. The source must remain valid as long as it is set on the mixer.
